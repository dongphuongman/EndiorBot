## Viewing

These are React prototypes that require a local HTTP server (CORS blocks file:// protocol):

```bash
# From the landing/ directory:
cd landing/endiorbot-landing && python3 -m http.server 8080
# Open http://localhost:8080

# App mockup:
cd landing/endiorbot-app && python3 -m http.server 8081
# Open http://localhost:8081/app.html

# SDLC Framework:
cd landing/sdlc-framework && python3 -m http.server 8082
# Open http://localhost:8082/SDLC%20Framework%20Landing.html
```

## Design Decisions

- **Aesthetic:** Terminal/hacker meets clean modern OSS — monospace accents on dark editorial
- **Color:** Near-black canvas + warm amber accent (distinctive against the indigo/green sea)
- **Typography:** Fraunces (display) + Inter Tight (body) + JetBrains Mono (code)
- **Bilingual:** EN/VI toggle, persisted to localStorage
- **Themes:** 4 palettes (Amber, Terminal-green, Cobalt, Paper-light)
