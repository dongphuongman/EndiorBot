## 5. Design Decisions

| Decision | Rationale |
|----------|-----------|
| Base profile only | ADR-056 explicitly scopes EndiorBot to Base; Governed profile is MTClaw concern |
| `schema_version` + `runtime_compat` | C2 split per ADR-056: manifest format version vs host runtime range |
| `name` from `project.id` | Already kebab-case normalized by existing `slugify()` |
| `description` fallback | Uses `project.description` when available; otherwise `${name} — Generated by EndiorBot` |
| STANDARD+ gate via `shouldCreateForTier()` | Reuses existing tier comparison logic |
| Seed READMEs only | Actual commands/skills are user-authored; we provide format documentation |
| `.claude-plugin/` NOT in `.gitignore` | Plugin manifest should be committed to repo |
